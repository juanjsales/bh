# Box & Health

Este projeto contém a Landing Page da "Box & Health".

## Estrutura do Projeto

- `index.html`: Arquivo principal da aplicação.
- `style.css`: Estilos personalizados adicionais.
- `src/LandingPage.tsx`: Componente React da página.
- `produto.jpg`: Foto principal do produto fornecida.

## Como Executar

O projeto utiliza Tailwind CSS via CDN. Basta abrir o arquivo `index.html` em seu navegador para visualizar a Landing Page.
